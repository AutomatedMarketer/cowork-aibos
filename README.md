# Claude Cowork AI Business Operating System (AIBOS)

> *Most AI templates ship 30 skills you can't relate to. AIBOS ships 7 skills every business needs, and teaches you to build the next 30 yourself — for your business, in your voice, on your terms.*

**Built by [Nuno Tavares](https://automatedmarketer.net) — Automated Marketer · VCInc**

---

## Install (two commands, 30 seconds)

In Cowork (Claude Desktop), open a new task and type:

```
/plugin marketplace add automatedmarketer/cowork-aibos
```

Then:

```
/plugin install cowork-aibos
```

That's it. AIBOS is now installed in your Cowork.

To start the install walkthrough, type:

```
start onboarding
```

Walk through 7 phases (60–90 minutes). At the end, your AI business operating system is live.

---

## What's inside

7 skills:

| Slash command | Plain English | What it does |
|---|---|---|
| `/onboard` | "start onboarding" | The 7-phase install. Resumable. |
| `/audit` | "check my system" | Scores your AIBOS on the 4 C's (0–100). |
| `/level-up` | "what should I improve next" | Recommends one specific move. |
| `/tune-up` | "weekly tune-up" | The 5-gate self-improvement loop. |
| `/morning-brief` | "morning brief" | Daily brief from inbox + calendar. |
| `/voice-writer` | "write this in my voice" | Drafts anything in your voice. |
| `/add-skill` | "build me a new skill" | Hands off to Cowork's Skill Creator. |

---

## Updates

When a new version ships:

```
/plugin update cowork-aibos
```

Your `about-me/`, `outputs/`, and `_aibos/state.md` survive — they're in your workspace, not the plugin.
